Fatcatlab Gesture Sensor

  1. Documentation
  2. Sensors
  3. Fatcatlab Gesture Sensor
This docs page is for kernel version 4.4.44-18-ev3dev-ev3

General Info

device_name fcl-gesture
Website fatcatlab.com
Connection UART/EV3
sysfs class lego-sensor
# Modes 4

Modes

mode Description units decimals1 num_values Values
GESTURE Gesture none 0 1 value0: Gesture (0 to 6)2
PROXIMITY Proximity none 0 1 value0: Voltage (0 to 127)
RGB-RAW Color none 0 3 value0: Red
value1: Green
value2: Blue
CLEAR3 Clear none 0 1 value0: Always 1

Commands

This sensor does not support commands.

Notes

  1. Decimal places. For example, if the range of a value is from 0 to 1000 and decimals is 1, then the actual range is 0.0 to 100.0 in the units specified. ↩

  2. Gesture Values:

    Value Description
    0 none
    1 left
    2 right
    3 up
    4 down
    5 near
    6 far

    ↩

  3. The CLEAR mode is used to reset the value of the GESTURE mode back to zero. ↩
