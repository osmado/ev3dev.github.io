Commands

command Description
CAL-WHITE Calibrate white
CAL-BLACK Calibrate black
SLEEP3 Put sensor to sleep
WAKE4 Wake up the sensor
60HZ Configures sensor for 60Hz electrical mains
50HZ Configures sensor for 50Hz electrical mains
UNIVERSAL Configures sensor for any (50/60Hz) electrical mains

Notes

  1. The address is programmable. See manufacturer documentation for more information. ↩

  2. Decimal places. For example, if the range of a value is from 0 to 1000 and decimals is 1, then the actual range is 0.0 to 100.0 in the units specified. ↩

  3. poll_ms must be set to 0 in order for sensor to sleep. ↩

  4. Will return an error (-ENXIO) if sensor is actually asleep. Completes successfully if sensor is already awake. ↩
