This is a technical reference about the things we have learned from working on the ev3dev kernel. It is mostly information about the hardware and device drivers.

EV3 Programmable Brick

Hardware

EV3 Intelligent Brick

  • Processor/System on Chip(SoC): Texas Instruments AM1808
    • 16550 UART
    • Serial Peripheral Interface (SPI)
    • Multimedia/Secure Digital Card Interface (MMC/SD)
    • Master/Slave I2C Interfaces
    • Programmable Realtime Unit (PRU)
    • USB 1.1 Host Port (OHCI)
    • USB 2.0 On-The-Go Port (OTG)
    • Real-Time Clock (RTC)
    • Pulse Width Modulators (PWM)
  • EEPROM: Microchip 24FC128
  • Display: 178x128 monochrome LCD
    • Sitronix ST7586 Controller/Driver
  • Analog/Digital Converter: Texas Instruments ADS7957
  • Motor Driver: Sanyo/ON Semiconductor LB1836M
  • Bluetooth: Panasonic PAN1325A-HCI-85
    • Texas Instruments CC2560A
  • Other:
    • Buttons
    • LEDs
    • Speaker
